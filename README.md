# 🚗 V2V Communication System

## 📌 Overview
This project simulates a **Vehicle-to-Vehicle (V2V) Communication System** using Python.  
It demonstrates how vehicles can exchange data to improve road safety and efficiency.

---

## 🎯 Objectives
- Simulate communication between vehicles
- Process and analyze transmitted data
- Demonstrate real-time decision making

---

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit (for UI)
